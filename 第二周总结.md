## 三、✨ 新增功能

### 3.1 考核配置页面（教师端）

**背景：** 第一周仅完成考核配置的后端 API，前端完全缺失配置入口和页面。

**实现内容：**

| 模块 | 文件 | 功能 |
|------|------|------|
| **配置页面** | `AssessmentConfigView.vue` | 完整的配置表单，含知识点标签输入、题型数量分配、评价维度（名称+权重）、及格分数 |
| **配置按钮** | `ChapterManageView.vue` | 章节管理表每行增加绿色"配置考核"按钮，点击跳转配置页 |
| **路由** | `router/index.ts` | 新增 `/admin/chapters/:id/assessment-config`（仅教师角色可访问） |
| **API** | `api/assessment.ts` | 新增 `getAssessmentConfig()` / `saveAssessmentConfig()` |

**页面功能细节：**
- 知识点：`el-select` 多选 + allow-create 模式，输入后回车添加标签
- 题型：三个 `el-input-number` 分别控制选择题/判断题/简答题数量，实时显示总题数
- 维度：动态行，可添加/删除，实时校验权重和 = 1.0
- 已有配置自动回填编辑

### 3.2 智能体 SSE 流式调用

**背景：** 第一周的智能体页面使用 setTimeout 模拟返回，非真实调用。

**修复：** `AgentInvokeView.vue` 接入 `createSSEConnection`，使用真实的 SSE 流式调用智能体，支持 `onUnmounted` 自动断开。

### 3.3 文档在线预览

**实现：**
- PDF：新标签页打开（浏览器原生预览）
- PPT/Word：触发下载

### 3.4 Docker 容器化部署（Phase 8 完成）

**第一周状态：** 未完成  
**本周完成：** ✅

| 组件 | 镜像 | 说明 |
|------|------|------|
| **backend** | `course-agent-backend` | Python 3.11 + FastAPI，启动时自动执行 `init_db` 初始化 |
| **frontend** | `course-agent-frontend` | Nginx 服务 Vue 构建产物 |
| **milvus** | `milvusdb/milvus:v2.4.14` | 向量数据库 |
| **milvus-etcd** | `quay.io/coreos/etcd:v3.5.17` | Milvus 元数据存储 |
| **milvus-minio** | `minio/minio:RELEASE.2024-11-07` | 向量数据持久化 |

**Docker 配置关键点：**
- Nginx 反代需配置 `proxy_buffering off;` 以支持 SSE 流式响应
- Nginx 使用 `set $backend_upstream "http://backend:8000";` 变量方式代理，实现运行时 DNS 解析
- Docker Compose 网络隔离为 `course-network`
- 后端 SQLite 挂载 volume 持久化

## 六、⚠️ 已知问题与限制
